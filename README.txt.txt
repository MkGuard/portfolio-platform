# Web Final Project - Portfolio Platform

---

## Project Overview

This project is a portfolio platform where users can register, log in, and manage their portfolio items. It includes secure user authentication, portfolio management, and optional Two-Factor Authentication (2FA) for enhanced security.

---

## Setup Instructions

### Prerequisites

1. Node.js: Ensure you have Node.js (version 16+) installed.
2. MongoDB Atlas: You need a MongoDB Atlas account for cloud database management.
3. Email Service: For email functionality, set up an account with a service like Nodemailer.

### Installation Steps

1. Clone the Repository:
   ```bash
   git clone https://github.com/MkGuard/portfolio-platform.git
   cd portfolio-platform

Install Dependencies: Run the following command to install necessary dependencies:

npm install

Run the Development Server: To start the app, run:

npm run dev

Access the Application: Open your browser and go to http://localhost:3000
